# LaTex_Template

This is a LaTex template used for daily study and note-taking

随手写了一个自己喜欢的LaTex笔记模板

## 使用方法：

文章内容在document.tex，

写完后直接编译main.tex即可

如若自己想要更改配置，修改template.tex即可

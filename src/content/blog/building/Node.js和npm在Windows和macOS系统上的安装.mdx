# Windows

### 壹 · 下载`node.js`的安装包
点击 [https://nodejs.org/dist/v22.16.0/node-v22.16.0-x64.msi](https://nodejs.org/dist/v22.16.0/node-v22.16.0-x64.msi)下载安装包

这里的链接是v22.16.0版本的，如果你需要安装其他版本，可以去[https://nodejs.org](https://nodejs.org/zh-cn)官网自行下载。

### 贰 · 安装并配置环境变量

运行安装包，根据提示完成安装

![图1-安装的目录](/figures/095-Node.js和npm在Windows和macOS系统上的安装.png)

![图2-自动添加环境变量](/figures/096-Node.js和npm在Windows和macOS系统上的安装.png)

安装完成后可以打开终端

```shell
node --version
npm --version
```
如果看到版本输出，即安装成功。

如果没有的话，可以根据下文检查系统环境变量是否正确添加。

### 其他 · 关于环境变量的添加

搜索栏搜索“环境变量”并打开，选择系统变量的`Path`

![图3-环境变量](/figures/097-Node.js和npm在Windows和macOS系统上的安装.png)

添加`node.js`的安装路径，即上文 图1 的目录

![图4-添加路径](/figures/098-Node.js和npm在Windows和macOS系统上的安装.png)

完成之后在终端输出版本号，即成功安装。

# macOS

这里推荐两种安装方式

## 一 · 通过Homebrew安装

如果你已经有了homebrew，这里可以直接安装

```zsh
brew install node 
```

## 二 · 手动下载安装

下载[https://nodejs.org/dist/v22.16.0/node-v22.16.0.pkg](https://nodejs.org/dist/v22.16.0/node-v22.16.0.pkg)并运行即可，
这里依然是特定版本，如果你需要其他版本请自行安装。
